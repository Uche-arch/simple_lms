# DevAcademy

Learn frontend development through structured roadmaps, curated resources, practical projects, and progress tracking.

Instead of jumping between random tutorials, DevAcademy provides a clear learning path that helps developers stay focused, build projects, and track their growth.

## Getting Started

Clone the repository:

```bash
git clone https://github.com/yourusername/devacademy.git
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

Open:

```text
http://localhost:3000
```
